# Slot-machine

### https://themdask.github.io/slot-machine/

Simple slot machine built using core Javascript/HTML5/CSS3. 

Work In Progress.

## Features
- 3 'spinners' with emojis
- spinning functionality
- betting functionality
- info modal with winning combinations
- autoplay checkbox
